# Changelog
All notable changes to StreamX Ultra are documented here.

## [v1.2] – Major Feature Update
### Added
- Glassmorphism-based futuristic Home UI
- Pinch-to-Zoom (YouTube-style) video scaling
- Aspect Ratio controls (Fit / Fill / Zoom)
- Smart LIVE badge for real-time streams
- Pull-to-refresh on Home Screen
- No Internet popup with retry mechanism

### Improved
- Player gesture smoothness (brightness & volume)
- Double-tap seek with animated indicators
- Search performance and instant play

---

## [v1.1] – First Stable Release
### Added
- Multi-platform authentication (Google, Facebook, GitHub)
- Redesigned authentication UI
- Rotation control & double-tap seek
- Live data usage & internet speed monitor
- Profile & account management
- Dedicated legal & disclaimer page

---

## [v1.0] – Initial Release
- First official public release
- Core Live TV & IPTV playback
- Navigation drawer & featured banner
- GitHub-based update checker
